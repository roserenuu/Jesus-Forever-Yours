# Manus Carousel Prompt Template

Use this as your base prompt when requesting a new carousel from Manus.
Swap out the bracketed fields for each new carousel.

---

Create an Instagram carousel for @jesusforeveryours with the following specs:

Topic: [TOPIC — e.g. "God sees your tears"]
Theme/Message: [1-2 sentence summary of the message]

DESIGN SPECS:
- Size: 1080 x 1440px (3:4 ratio)
- Font: Playfair Display Bold Italic — EXACT same weight and style as the "Trust His Timing" carousel. This must never change.
- Text color: [COLOR + HEX — must not be a color already used. See carousel-log.txt]
- BOOK accent color: [COLOR + HEX — must not be an accent already used. See carousel-log.txt]
- Text: centered, large, 1-2 lines per slide, clean and breathable

STRUCTURE:
- Slide 1: Hook line 1 (see hook-bank.md for unused hooks)
- Slide 2: Hook line 2 (different angle, same theme)
- Slides 3-13: Love letter from God content (1-2 lines per slide, flows as a story)
- Slide 13: "Forever Yours, Heavenly Father" sign-off
- Slide 14: Book promo slide — feature book cover image, CTA to comment BOOK

CONTENT RULES:
- Written as a love letter from God to the reader
- Warm, intimate, encouraging tone
- No @ handles on any slide
- No emojis
- Pull from this content if relevant: [PASTE RELEVANT LINES OR LEAVE BLANK]

CAPTION (include with delivery):
- Short, meaningful, ties to the theme
- CTA: comment BOOK
- No hashtags, no emojis

Save as: carousels/[NUMBER]-[topic-slug]/slide_1.png through slide_14.png + caption.txt
