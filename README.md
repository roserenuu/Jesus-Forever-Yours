# Jesus Forever Yours

**Content Hub for the Jesus Forever Yours Brand by Rose Renuu**

This repository is the central content library for the **Jesus Forever Yours** brand, home to all carousel designs, reel scripts, captions, and creative documentation for the Instagram brand page [@jesusforeveryours](https://instagram.com/jesusforeveryours) and the main creator page [@roserenuu](https://instagram.com/roserenuu).

The brand centers around the book **"Forever Yours: 24 Days With Jesus"** — a 24-day devotional written as love letters from God. Each day includes a letter beginning with "My Child" and ending with "Forever Yours, Heavenly Father," paired with a scripture verse. The book is available in hardcover and eBook at [jesusforeveryours.com](https://jesusforeveryours.com).

## Carousels (15 Total, 210 Slides)

Each carousel folder contains **14 slides** (1080 x 1440px, 3:4 ratio) and a **caption.txt** file with the ready-to-post Instagram caption. All carousels are designed for the [@jesusforeveryours](https://instagram.com/jesusforeveryours) brand page.

## Reel Scripts (24 Days)

The file `reel-scripts/all_24_reel_scripts.md` contains all **24 reel scripts** for the "Powerful Scriptures You Should Know" series on [@roserenuu](https://instagram.com/roserenuu). Each script follows the format:

1. Scripture verse read aloud
2. Book, chapter, and verse reference
3. "This is what God is telling you today" love letter message
4. "Forever yours, heavenly father" sign-off
5. Personal closing from Rose
6. Ready-to-post caption with BOOK CTA

The scripts cover themes from God's sacrifice and trust, to healing, self-love, rest, peace, strength, protection, provision, hope, identity, courage, faithfulness, patience, new beginnings, purpose, freedom, and unconditional love.
